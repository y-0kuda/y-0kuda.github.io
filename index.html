イントロダクション

MyDocs へようこそ。このドキュメントでは、フレームワークの基本的な使い方からアドバンスドな機能まで、丁寧に解説します。

概要

MyDocs はモダンな Web アプリケーション開発のための包括的なフレームワークです。シンプルなコンセプトに基づきながら、プロダクションレベルの機能を提供します。

💡 初めての方はクイックスタートから始めることをお勧めします。

特徴

  • 軽量 — コア部分はわずか 12KB (gzip 圧縮後)
  • 型安全 — TypeScript ファーストの設計
  • ゼロ依存 — 外部ライブラリへの依存なし
  • プラグイン対応 — 柔軟なプラグインシステム

基本的な使い方

CDN 経由で即座に利用できます。

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <script src="https://cdn.example.com/mydocs.min.js"></script>
</head>
<body>
  <div id="app"></div>
  <script>
    MyDocs.init('#app');
  </script>
</body>
</html>

次のステップ

クイックスタートガイドに進んで、最初のプロジェクトを作成しましょう。

クイックスタート

5分で最初のアプリを動かしましょう。

Step 1: インストール

npm install mydocs

Step 2: 初期化

import { MyDocs } from 'mydocs';

const app = new MyDocs({
  target: '#app',
  theme: 'light',
  debug: false,
});

Step 3: 動作確認

ブラウザで http://localhost:3000 を開いてください。

🎉 おめでとうございます！最初のアプリが完成しました。

ダークモード NEW

OS 設定と連動したダークモードをサポートします。

自動切り替え

prefers-color-scheme メディアクエリを使って、OS の設定に自動的に追従します。 

@media (prefers-color-scheme: dark) {
  :root {
    --color-bg: #1a1b1e;
    --color-text: #c9d1d9;
  }
}

手動切り替え

MyDocs.setTheme('dark');  // ダーク
MyDocs.setTheme('light'); // ライト
MyDocs.setTheme('auto');  // OS 設定に追従

API リファレンス

全 API の詳細なリファレンスです。

MyDocs.init(selector, options)

アプリケーションを初期化します。

パラメータ 型 説明
selector string マウント先の CSS セレクタ
options Options 設定オブジェクト（任意）

MyDocs.setTheme(theme)

テーマを変更します。

MyDocs.setTheme('dark' | 'light' | 'auto')

MyDocs.destroy()

インスタンスを破棄し、イベントリスナーを解除します。
